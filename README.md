# GestureRemote

> A wearable gesture-based remote control system using STM32 and an IMU sensor to control electronic devices through natural hand movements.

## 📖 Overview

GestureRemote is a wearable remote that detects hand gestures and wirelessly sends commands to compatible devices. It aims to provide an intuitive and portable alternative to traditional remote controls.

## ✨ Features

- Wearable design
- Gesture recognition using IMU
- Wireless communication
- Low power consumption
- Expandable for multiple applications

## 🛠️ Hardware

- STM32 Microcontroller
- IMU Sensor (MPU6050/ICM-20948)
- Wireless Module (TBD)
- Battery
- Push Button
- LEDs
